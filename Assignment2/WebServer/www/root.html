You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a JSON object from the /random request
  • /random -- returns index.html
  • /multiply?num1=3&num2=4 -- multiplies num1 and num2 and returns the result
  • /github?query=users/amehlhase316/repos -- returns information about the public repositories of the given GitHub user

New Request Types

1. Currency Conversion:

This request converts an amount from one currency to another using predefined exchange rates.

Arguments:

  • amount: The amount to be converted (e.g., 100).
  • from: The currency to convert from (e.g., USD).
  • to: The currency to convert to (e.g., EUR).

Example URL: http://localhost:9000/convertCurrency?amount=100&from=USD&to=EUR

Example Data:

  Converted amount: 85.0 EUR

2. Greeting Message:

This request generates a personalized greeting message based on the user's name and the specified language.

Arguments:

  • name: The name of the person to greet (e.g., John).
  • language: The language for the greeting (e.g., English, Spanish, French).

Example URL: http://localhost:9000/greet?name=John&language=Spanish

Example Data:

  Hola, John!
